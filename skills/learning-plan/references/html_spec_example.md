# app_0102_01 - 排序算法步骤模拟器

## 基本信息

- **应用 ID：** app_0102_01
- **所属阶段：** 第一阶段：数据结构基础 > 排序算法
- **关联知识点：** 冒泡排序、选择排序、插入排序
- **交互类型：** 步骤模拟器

## 功能目标

帮助用户直观理解三种基础排序算法的执行过程。用户可以逐步执行算法，观察每一步中数组元素的比较和交换，建立对排序算法时间复杂度的直觉认知。

## 内容数据

- 三种排序算法：冒泡排序、选择排序、插入排序
- 默认数组：`[38, 27, 43, 3, 9, 82, 10]`
- 支持用户自定义数组（最少 3 个元素，最多 15 个，数值范围 1-999）
- 每种算法的伪代码文本

## 功能逻辑

### 核心交互流程

1. 用户进入页面，看到默认数组的可视化柱状图
2. 顶部有算法选择器（冒泡/选择/插入），默认选中冒泡排序
3. 用户点击"下一步"按钮，算法执行一步操作
4. 当前正在比较的两个元素高亮（橙色），已排好序的元素变为绿色
5. 右侧面板同步显示伪代码，当前执行行高亮
6. 底部显示操作日志："比较 arr[0]=38 和 arr[1]=27 → 交换"
7. 用户可以点击"自动播放"，以可调速度自动执行
8. 排序完成后显示统计：比较次数、交换次数、总步数

### 功能细节

- 算法切换时重置数组到初始状态
- "重置"按钮恢复初始数组
- "随机生成"按钮生成新的随机数组
- "自定义"按钮允许用户输入自己的数组
- 自动播放速度可调：慢（1s/步）、中（0.5s/步）、快（0.2s/步）
- 支持"上一步"回退操作

### 状态管理

- 当前算法类型
- 数组当前状态（每一步的快照）
- 当前步骤索引
- 比较次数、交换次数计数器
- 自动播放状态（播放中/暂停）
- 播放速度

## 界面设计

### 布局

```
┌─────────────────────────────────────────────┐
│  [冒泡排序] [选择排序] [插入排序]    算法选择  │
├──────────────────────┬──────────────────────┤
│                      │  伪代码面板           │
│   数组可视化区域      │  ┌──────────────┐    │
│   (柱状图)           │  │ for i = 0... │    │
│                      │  │ > if a[i]... │←当前│
│                      │  │   swap(...)  │    │
│                      │  └──────────────┘    │
├──────────────────────┴──────────────────────┤
│  操作日志：比较 arr[0]=38 和 arr[1]=27 → 交换  │
├─────────────────────────────────────────────┤
│  [上一步] [下一步] [▶自动] [速度:中]          │
│  [重置] [随机生成] [自定义]                    │
├─────────────────────────────────────────────┤
│  统计：比较 12 次 | 交换 8 次 | 步骤 3/28     │
└─────────────────────────────────────────────┘
```

### 视觉风格

- **色调：** 清新学术风，主色 #4A90D9（蓝），辅色 #7ED321（绿）、#F5A623（橙）
- **字体建议：** 正文 Inter/Noto Sans SC，伪代码 JetBrains Mono
- **动画：** 元素交换时平滑过渡（300ms ease-in-out），高亮切换淡入淡出

### 响应式要求

- 最小支持宽度 375px（移动端竖屏）
- 移动端伪代码面板折叠到底部，可展开

## 交互细节

- **输入方式：** 点击按钮、点击算法标签切换
- **反馈机制：** 比较时元素闪烁高亮，交换时元素平滑移动动画，排序完成时全部元素依次变绿并弹跳
- **进度指示：** 底部步骤计数器 "步骤 3/28"
- **完成条件：** 数组完全有序，所有元素变为绿色

## 数据接口

### 输入数据格式

```json
{
  "algorithms": ["bubble", "selection", "insertion"],
  "default_array": [38, 27, 43, 3, 9, 82, 10],
  "pseudocode": {
    "bubble": "for i = 0 to n-1:\n  for j = 0 to n-i-2:\n    if arr[j] > arr[j+1]:\n      swap(arr[j], arr[j+1])",
    "selection": "...",
    "insertion": "..."
  }
}
```

### 输出数据格式

```json
{
  "app_id": "app_0102_01",
  "completed": true,
  "result": {
    "algorithms_explored": ["bubble", "selection"],
    "total_steps_executed": 45,
    "time_spent_seconds": 320,
    "custom_arrays_tried": 2
  }
}
```
